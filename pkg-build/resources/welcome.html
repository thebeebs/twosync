Welcome to twosync

twosync keeps two folders in sync automatically — adding, updating, and deleting files in both directions.

After installation you'll be able to:

  • Run a one-off sync from Terminal: twosync ~/FolderA ~/FolderB
  • Set up hourly automatic sync: twosync-setup ~/FolderA ~/FolderB
  • View logs: tail -f ~/.twosync/twosync.log

Requires macOS 11 or later with Python 3 (included with macOS).
